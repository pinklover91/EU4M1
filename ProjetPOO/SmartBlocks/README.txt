                      SmartBlocks 2D Physics Engine
=========================================================================

2D Physics engine used to simulate the behavior of Moving Objects of any 
shape, under the forces generated by a smart surface composed of blocks.
More information at: http://www.smartsurface.cnrs.fr/

All the necessary information about this development can be found in the
corresponding javadoc: dist/javadoc/index.html

This project was generated using Netbeans IDE 6.9.1 using the Java SE
platform and should work as well in newer versions and reasonably older 
versions.
